# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 2.x     | :white_check_mark: |
| 1.x     | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by opening an issue at
https://github.com/marcuwynu23/Auto/issues with the label `security`.

Do **not** open a public issue for critical vulnerabilities. Instead, email the
maintainer directly or use GitHub's private vulnerability reporting feature.

We will acknowledge receipt within 48 hours and provide a timeline for a fix.
